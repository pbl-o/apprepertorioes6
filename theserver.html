𝄞 Mi repertorio - Server 𝄞

  • GET/http://localhost:3000/canciones -> Muestra las canciones agregadas
  • POST/http://localhost:3000/canciones -> Permite agregar una canción
  • PUT/http://localhost:3000/canciones/:id -> Permite actualizar los datos de una canción (se requiere un id)
  • DELETE/http://localhost:3000/canciones/:id -> Permite borrar una canción (se requiere un id)

𝄞 Como utilzar con herramientas para testear API 𝄞

  • GET/http://localhost:3000/canciones -> Ejecutar test.
    • POST/http://localhost:3000/canciones

    • Ingresar los datos requeridos a continuación y ejecutar test posteriormente.

      {
      "id": "[coloque un ID]",
      "titulo" : "[colocar nombre de canción]",
      "artista": "[colocar nombre de artista]",
      "tono":"[elegir una tonalidad]"
      }

  • PUT/http://localhost:3000/canciones/:id -> Elegir un Id de los ya existentes en el archivo json, alterar alguna de las informaciones (id,titulo, artista o tono) y ejecutar test.
  • DELETE/http://localhost:3000/canciones/:id -> Elegir un Id y ejecutar test.
